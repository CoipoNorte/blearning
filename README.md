# 🅱️ blearning

Guía interactiva para aprender React-Bootstrap desde cero. Landing page con 14 secciones que cubren todos los componentes principales de React-Bootstrap 5 con demos en vivo y código copiable.

![React-Bootstrap](https://img.shields.io/badge/React--Bootstrap-5-7952b3?logo=bootstrap)
![React](https://img.shields.io/badge/React-18-61dafb?logo=react)
![Vite](https://img.shields.io/badge/Vite-5-646cff?logo=vite)

## 🔗 Temas cubiertos

Setup · Grid · Tipografía · Buttons · Cards · Forms · Navs y Tabs · Modals · Alerts y Badges · Tables · Accordion · Progress y Spinners · Toasts y Offcanvas · Componentes combinados

## 🛠️ Tecnologías

- **React-Bootstrap** — Componentes Bootstrap 5 para React
- **Bootstrap 5** — Framework CSS
- **React 18** — Biblioteca de UI
- **Vite** — Build tool
- **Lucide React** — Iconos

## 🚀 Ejecutar

```bash
npm install
npm run dev
```
